# oci-infra

Projetos de infraestrutura na Oracle Cloud Infrastructure (OCI) usando Terraform e Ansible.

## Projetos Terraform

- **oci-bucket** — Bucket de Object Storage
- **oci-instance-cloudinit** — Instância com Cloud-Init
- **oci-loadbalancer-vms** — Load Balancer com múltiplas VMs
- **vcn-instance** — VCN + Instância base

## Ansible

- Playbooks para configuração de instâncias OCI

## OCI Específico

- **oci-auto-setup** — Setup automático com Terraform + Cloud-Init
- **oci-zabbix-agent** — Instalação automatizada do Zabbix Agent
